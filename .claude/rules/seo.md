# SEO Technique

## Le site d'une agence SEO doit etre irreprochable en SEO technique

- **SSR/SSG**: Pages statiques en SSG (landing, history), pages dynamiques en SSR (report)
- **Meta tags dynamiques**: Title, description, OG tags uniques par page
- **Structured Data**: JSON-LD pour Organization, WebApplication, BreadcrumbList
- **Sitemap**: `sitemap.xml` auto-genere
- **Robots.txt**: Correctement configure
- **Canonical URLs**: Sur chaque page
- **Performance**: Lighthouse 90+ sur les 4 metriques (Performance, Accessibility, Best Practices, SEO)
- **Core Web Vitals**: LCP < 2.5s, FID < 100ms, CLS < 0.1
- **Images**: Format WebP/AVIF, lazy loading, dimensions explicites
- **Semantic HTML**: Headings hierarchy correcte (h1 unique par page), landmarks, aria labels
- **Mobile-friendly**: Landing page responsive, viewport meta tag
- **Open Graph**: og:title, og:description, og:image pour le partage social
- **Favicon**: Multi-format (ico, png, apple-touch-icon)
